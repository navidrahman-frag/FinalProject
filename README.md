# FinalProject
# Course Management System

A desktop-based **Course Management System** developed using **C# Windows Forms (.NET Framework)** and **SQL Server**.  
This project helps manage students, teachers, courses, enrollments, payments, and academic activities through an interactive graphical user interface.

---

# Features

- User Login & Authentication
- Student Management
- Teacher Management
- Course Management
- Student Enrollment System
- Course Assignment
- Payment Management
- Dashboard Interface
- SQL Server Database Integration
- DataGridView Based Data Display
- CRUD Operations (Create, Read, Update, Delete)

---

# Technologies Used

- C#
- Windows Forms (.NET Framework)
- SQL Server
- ADO.NET
- Visual Studio

# How To Run The Project

1. Clone or download the repository.
2. Open `FinalProject.slnx` using Visual Studio.
3. Open SQL Server Management Studio.
4. Run the `script.sql` file to create the database and tables.
5. Update the SQL Server connection string in `App.config`.
6. Build and run the project.
